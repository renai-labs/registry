# Attribution

This skill applies the **Diátaxis** documentation framework.

- **Framework:** Diátaxis — a systematic approach to technical documentation authoring.
- **Author:** Daniele Procida
- **Source:** https://diataxis.fr
- **License:** The Diátaxis framework and its documentation are licensed under
  [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/).

The framework organizes documentation into four distinct types — tutorials,
how-to guides, reference, and explanation — based on user needs. All credit for
the underlying methodology belongs to Daniele Procida and the Diátaxis project.
